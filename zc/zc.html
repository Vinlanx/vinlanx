Zone Conquest

Attack. Defend. Dominate.

A structured, team-based objective mode where Red Attackers must conquer zones sequentially while Blue Defenders hold the line.

Important Capture Rules

1. Tug of War: One shared progress bar (Red ↔ Neutral ↔ Blue).

2. Sequential Lock: Red must capture points in order (1 → 2 → 3). Once fully captured, a point is LOCKED and cannot be retaken by Blue.

3. Empty Point: Progress freezes if no players are inside the zone.

Operational Overview

Zone Conquest is a competitive mode where teamwork, positioning, and space control matter more than raw combat.

The outcome depends on maintaining a player advantage inside capture zones. More allies equals faster capture. If numbers are equal, progress pauses.

Win Conditions: Red wins by capturing all points. Blue wins if the match timer expires.

Game States & Rules

Match Cycle

  • IDLE: Match not running (Grey).
  • LOBBY WAIT: Waiting for players after start (White).
  • WAITING: Pre-start phase. Red to waiting room, Blue to base (Yellow).
  • PLAYING: Active combat and capturing (Green).

Respawn System

  • If you die during PLAYING, you are sent to the Wait Room.
  • After 30 seconds, you redeploy to your Team Base.
  • Respawn timers are handled server-side.

Economy (Optional)

  • Supports Lightman’s Currency.
  • Owning teams receive periodic coin payouts.
  • Admins can toggle payouts for tournament play.

Admin Field Manual

1. Zone Creation: Use /setpointzc to define rectangular capture areas (IDs 1-20).

2. Infrastructure: Set up Bases, Waiting Rooms, and the Lobby using location commands.

3. Teams: Create vanilla teams with "Red" or "Blue" in the name. Add players via /team join.

Player Keybinds

  • O : Open Info Screen
  • TAB (Hold) : View Match Timer

Command Terminal

Requires OP Permission Level ≥ 2

Zone Management

/setpointzc <id> <w> <l> <time> Create a point. ID: 1-20, Time in seconds.
/listpointszc Open interactive list to Teleport or Delete points.

Match Control

/startgame [minutes] Initiate match (default 30 mins).
/stopgame [returnplayers] Stop match and reset timer.

Locations

/setbasered | /setbaseblue Set team spawn bases.
/setwaitroomred | /setwaitroomblue Set death/waiting rooms.

F.A.Q.

Q: How are teams detected?
A: By vanilla team name. Any name containing "red" or "blue" (case-insensitive) is automatically assigned.
Q: Can I disable captures temporarily?
A: Yes, use /capturestatuszc stop and /capturestatuszc resume.
Q: Does the world remember game state?
A: Yes. The mod creates zoneconquest_autostart.json. If the server restarts during a match, it will attempt to resume if IDLE.
